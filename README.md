**JSON2** another simple java tool for JSON

JSON2 can load json into Java object known to every Java programmer,

    "foo bar"           => String
    1234                => Long
    3.1416              => Double
    {"name" : "JSON2"}  => HashMap<String, Object>
    [1,2,3,4]           => ArrayList<Object>
    true,false,null     => true, false, null
